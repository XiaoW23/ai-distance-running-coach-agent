《丹尼尔斯经典跑步训练法》核心总结

一、 跑步训练的五种基本强度（按配速由慢到快）：
1. E（Easy）轻松跑：心率最大值的65%-79%。目的是建立耐力基础，增强心肌，增加肌肉中的线粒体和毛细血管密度。通常占总跑量的75%-80%。如果感觉疲劳，应增加E跑比例或完全休息。
2. M（Marathon）马拉松配速跑：心率最大值的80%-90%。目的是模拟比赛配速，适应马拉松比赛的心理和生理压力。
3. T（Threshold）乳酸阈值跑：心率最大值的88%-92%。目的是提高身体清除乳酸的能力，提升耐力极限。配速通常是跑者刚好能维持60分钟的配速。
4. I（Interval）间歇跑：心率最大值的98%-100%。目的是最大化摄氧量（VO2max）。典型的I跑每次持续3-5分钟，中间包含等时间或略短的恢复慢跑。
5. R（Repetition）重复跑：目的是提高跑步经济性和速度。距离较短（通常不超过2分钟），强调充足的恢复时间，跑姿要流畅。

二、 训练周期化原则
1. 基础期：以E跑为主，打好有氧基础，可以加入少量短距离冲刺以维持肌肉弹性。
2. 进展期：引入重复跑(R)和少量间歇跑(I)，提升肌肉力量和速度。
3. 巅峰期：强调间歇跑(I)和乳酸阈值跑(T)，将身体状态调整至最佳。
4. 竞赛期：减少训练总量（减量 taper），保持强度，让身体恢复疲劳，迎接比赛。

三、 负荷与恢复
- 当连续两天的训练心率偏高、配速下降时，说明身体处于过度疲劳状态，需要安排完全休息或极慢的恢复跑（E跑的下限）。
- 每次高强度训练（T, I, R）后，必须安排至少一天的轻松跑(E)或休息。
- 周跑量的增加不应超过前一星期的10%。


【核心强度训练量化约束（Agent 拦截规则）】
Agent 在评估用户的课表（如 CSV 数据）时，必须严格应用以下负荷上限，若用户超标，必须发出警告：

# 1. T 跑（乳酸阈值跑）约束：
* 连续节奏跑（Tempo Run）：单次时间上限为 20 分钟。
* 巡航间歇跑（Cruise Intervals）：单次总跑量不得超过周跑量的 10%。组间休息比例必须极短，通常为 跑:歇 = 5:1（例如跑 5 分钟休息 1 分钟）。

# 2. I 跑（VO2max 间歇跑）约束：
* 单次总跑量上限：周跑量的 8% 或 10 公里（取两者中较小值）。
* 单组持续时间上限：绝对不能超过 5 分钟。
* 休息与恢复规则：跑与休息的时间比例应等于或略小于 1:1（如跑 3 分钟，休息 2-3 分钟）。休息期心率必须降至最大心率的 65%-70% 才能开启下一组。

# 3. R 跑（经济性重复跑）约束：
* 单次总跑量上限：周跑量的 5% 或 8 公里（取两者中较小值）。
* 单组持续距离上限：单组最多持续 2 分钟（通常为 200m - 600m）。
* 休息与恢复规则：必须完全恢复！跑与休息的时间比例通常为 1:2 或 1:3。若用户在 R 跑中休息时间极短（类似 I 跑的 1:1），Agent 需警告用户正在将 R 跑错误地跑成无氧耐酸训练。


【800m/1500m 专项高阶配速：FR (快速重复跑)】

# 1. FR 定义与目的（提供给 Agent 的概念锚点）
* FR 用于中跑训练的巅峰期与竞赛期。
* 核心目的是模拟 800m 比赛配速，提高特定发力下的机械效率，激活快肌纤维，非绝对极限冲刺。

# 2. FR 核心配速映射（截取目标区间）
当 800m 当前水平/目标为 2:20 时：
- FR 200m 规定用时：35秒
- FR 400m 规定用时：70秒

当 800m 当前水平/目标为 2:10 时：
- FR 200m 规定用时：32秒
- FR 400m 规定用时：65秒

# 3. ⚠️ Agent 系统拦截规则（强制覆盖）
* 规则 A：若用户执行 200m 训练，用时在 FR 规定区间内（如 32s-35s），且组间休息较短（如 2 分钟内），Agent 应判定为【Daniels FR 专项耐酸与经济性训练】。
* 规则 B：若用户穿戴钉鞋执行 200m 或更短距离训练，单组用时大幅快于 FR 规定时间（如 200m 跑进 30s 内），且组间休息极长（4-5分钟及以上），Agent 必须将其剥离出 Daniels 评价体系，独立标记为【体系外：绝对极速与神经募集模块】，禁止输出“超速/过度训练”等错误警告。


【800m/1500m 专项训练周期化规则（4 阶段框架）】
Agent 在为用户制定或评估宏观计划时，必须遵循以下中跑专属骨架：

* Phase I（基础期，前 25% 周期）：
  核心全为 E 跑与 L（长距离）跑。唯一允许的速度刺激是轻松跑后的“快步跑（Strides）”，严禁加入系统性 I 跑。
* Phase II（进阶期，25%-50% 周期）：
  引入核心武器：R 跑（重复跑）或 FR 跑。此阶段重点在于通过极短的 R/FR 提升跑步经济性和 800m 肌肉募集能力，建立速度基础。继续保持 E 跑，暂不引入 I 跑。
* Phase III（巅峰期，50%-75% 周期）：
  训练最艰苦的阶段。保留少量 R 跑维持速度，全面引入 I 跑（间歇跑）以最大化压榨 VO2max。典型的硬核周包含一次 I 跑课表和一次长距离课表。
* Phase IV（竞赛期，最后 25% 周期）：
  开始减量（Taper）。削减 I 跑的组数和总周跑量，引入 T 跑（乳酸阈值跑）以维持有氧状态，利用少量 FR 跑模拟比赛配速（Race Pace）。


【Agent 配速评估的弹性宽容度（针对中距离跑者）】
注意：800m/1500m 跑者通常具备较强的无氧速度储备，但有氧耐酸能力可能相对滞后。
* 判定规则：当 Agent 根据用户的 1500m 成绩查出其 VDOT（如 VDOT=54）时，在实际评估其 T 跑和 I 跑的执行配速时，允许存在 1-2 个 VDOT 等级的“降级宽容度”（即允许其 T/I 实际配速掉落至 VDOT 53 或 52 的标准）。
* 心率优先原则：若用户的 E 跑配速低于 VDOT 规定区间，但心率已达到上限（如 150-160 bpm），Agent 应判定为【合理执行】，严禁催促用户提速。
